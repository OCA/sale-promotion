This module adds a new option in promotion/coupon programs that allows to
automatically add free products from promotion/coupon programs.

**Table of contents**

.. contents::
   :local:

Usage
=====

Activation is automatic.
To make a promotion/coupon program automatically adds free products, 
you have to check the automatically adds free products checkbox in the
promotion/coupon program form.
